1. Installation : 

> git clone https://github.com/140ch204/20_the_gossip_project.git
>
> cd 20_the_gossip_project
>
> bundle install
>
>rails db:drop
> 
> rails db:create
>
> rails db:migrate
> 
> rails db:seed
>
> rails s
>

Dans le navigateur aller à l'URL suivante : 

http://localhost:3000/

Enjoy !
