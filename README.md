<h2> THP 21/11/2019 </h2>

1. Installation : 

> git clone https://github.com/140ch204/38_ajax2
>
> cd 38_ajax2
>
> bundle install
>
> rails db:drop
> 
> rails db:create
>
> rails db:migrate
> 
> rails db:seed
>
> rails s
>

Dans le navigateur aller à l'URL suivante : 

http://localhost:3000/

Enjoy !

2. Connaître les versions utilisées : 

>
> rails about
